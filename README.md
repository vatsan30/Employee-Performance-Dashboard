# 📊 Employee Performance Tracker | HR Analytics Dashboard

## 🚀 Project Overview
This project focuses on building an end-to-end HR analytics solution to analyze workforce data and generate actionable insights for business decision-making. The dashboard provides visibility into employee distribution, compensation patterns, diversity metrics, and location-based cost analysis.

---

## 🎯 Problem Statement
Organizations often struggle to get a unified view of workforce data spread across multiple systems such as employee records, salary data, and location information.

This leads to:
- Lack of visibility into workforce distribution
- Difficulty in identifying gender pay gaps
- Inefficient salary benchmarking across departments
- Poor decision-making in hiring and cost optimization

👉 The goal of this project is to consolidate data and build an interactive dashboard that helps stakeholders make data-driven HR decisions.

---

## 🧠 Objectives
- Analyze workforce distribution across departments, locations, and job roles  
- Identify compensation trends and salary benchmarks  
- Evaluate gender diversity and pay gap  
- Provide insights for cost optimization and workforce planning  

---

## 🛠️ Tools & Technologies Used
- **SQL** → Data cleaning, transformation, and exploratory data analysis (EDA)  
- **Power BI** → Data modeling, DAX measures, and dashboard visualization  
- **Excel/CSV** → Initial data handling  

---

## 🔄 Steps Taken

### 1️⃣ Data Understanding & Cleaning
- Validated data across multiple tables (employee, salary, location)
- Checked for null values, duplicates, and inconsistencies
- Standardized categorical fields (department, country, etc.)

---

### 2️⃣ Data Integration
- Joined datasets using `employee_id`
- Created a unified employee master dataset combining:
  - Demographics
  - Salary details
  - Location data

---

### 3️⃣ Exploratory Data Analysis (SQL)
- Performed headcount analysis across departments and countries
- Analyzed gender distribution and diversity metrics
- Calculated average, min, and max salary
- Identified salary distribution patterns and bonus trends
- Evaluated gender pay gap across departments

---

### 4️⃣ Data Modeling (Power BI)
- Built a structured data model for analysis
- Created relationships between datasets
- Designed calculated measures using DAX:
  - Total Employees
  - Average Salary
  - Gender %
  - Gender Pay Gap

---

### 5️⃣ Dashboard Development
Developed an interactive multi-page dashboard with:
- KPI cards (Headcount, Salary, Diversity)
- Department-wise employee distribution
- Salary analysis by department and job role
- Gender distribution visualization
- Geographic salary insights (Map view)
- Age demographics analysis

---

## 📊 Key Insights
- IT and Sales departments have the highest workforce concentration  
- Salary distribution varies significantly across departments  
- Gender ratio is relatively balanced but slight disparities exist  
- Certain locations contribute more to total salary cost  
- Bonus allocation differs across departments indicating varied incentive structures  

---

## 📈 Dashboard Features
- Interactive filters (Country, Department, Gender)
- Drill-down capability for detailed analysis
- Multi-page structured reporting
- Clean and intuitive UI for business users

---

## 🧩 Business Impact
- Enables HR teams to make data-driven hiring decisions  
- Helps identify pay gaps and improve compensation fairness  
- Supports leadership in cost optimization strategies  
- Improves workforce planning and resource allocation
